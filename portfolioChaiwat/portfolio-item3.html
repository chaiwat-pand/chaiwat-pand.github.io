ExerciseTracker

A Back-end Application

This backend application was originally designed to be used with another project of mine, FitnessExpert. In this project, once connected with MongoDB, we can send requests and get responses back from the server.

All of the CRUD operations; Create Read Update Delete; are used in this project. This project can be downloaded and used by anyone. However, you will notice that in "config" folder there is a file named "db.js". You will find variables that aren't declared therefore you will have to create those variables in a separate ".env" file.

The variables that needs to be declared are the following: NODE_ENV = development PORT = 8000 (or any available port that you want to use) MONGO_URI = This link will be unique since it is specific to every individual for security reasons. You can find it after signing in to the MongoDB account and choosing "Connect to Sandbox".
