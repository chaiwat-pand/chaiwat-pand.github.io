FitnessTracker

An Exercise Tracker

The FitnessTracker is single webpage application allowing people to record their exercise activities. Users are able to store their exercise details, update specific details of any given exercise, and delete the ones that are no longer needed.

The front-end part of the Reality is created using React, a popular JavaScript library for building user interfaces based on UI components. As for the back-end, it was created using Node.js and Express.js.

In order for users to record their exercise activities, they will first need to sign-up and then log-in first. Once they have logged in, they will then be able to update both their personal profile and activities information.

All of the user's data will be stored on MongoDB, a NoSQL database program which uses JSON-like documents. The front-end and back-end were connected with a JavaScript library known as Axios. In this project, Axios was used to make HTTP requests and responses from Node.js.
