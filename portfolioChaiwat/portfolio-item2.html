FitnessExpert

A Fitness Brand Webpage

FitnessExpert is the first webpage I ever created. It is fictional platform dedicated to improving fitness level. Once you enter the webpage, you will find different fitness program that people can enroll in. Each program is designed for different purposes.

At the bottom of the page, there is also a form where users can fill out the details of their exercise sessions. One important thing to note about FitnessExpert is that it is a responsive webpage, meaning it works on any devices no matter their screen size.

This project was created utilizing HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) to create and design the UI interfaces.
